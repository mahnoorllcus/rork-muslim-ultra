# Improve Quran audio controls and stop playback when leaving the page

**Features**

- [x] Add a compact playback bar on the Quran page with previous, play/pause, and next controls.
- [x] Let the controls switch between surahs while keeping the reading screen easy to use.
- [x] Stop the recitation automatically when the user leaves or closes the Quran screen.
- [x] Keep the current surah title centered cleanly while audio controls stay neatly positioned.
- [x] Preserve a reliable listening experience so play, pause, previous, and next behave consistently.

**Design**

- [x] Use a small polished audio control row that feels native to mobile, with clear touch targets and balanced spacing.
- [x] Keep the surah name visually dominant in the center, with smaller secondary controls around it.
- [x] Match the existing app theme so the new player feels built-in rather than added on.
- [x] Show clear playback state so users can instantly see whether recitation is playing or paused.

**Pages / Screens**

- [x] **Quran page**: shows the improved playback controls, supports previous and next surah navigation during listening, and stops audio when the page is closed or left.
