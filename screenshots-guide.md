# App Store Screenshots Guide

## Required Screenshots

### iPhone Screenshots (Required)
- **6.7" Display (iPhone 14 Pro Max)**: 1290 x 2796 pixels
- **6.5" Display (iPhone 11 Pro Max)**: 1242 x 2688 pixels  
- **5.5" Display (iPhone 8 Plus)**: 1242 x 2208 pixels

### iPad Screenshots (Optional but Recommended)
- **12.9" Display (iPad Pro)**: 2048 x 2732 pixels
- **11" Display (iPad Pro)**: 1668 x 2388 pixels

### Android Screenshots
- **Phone**: 1080 x 1920 pixels minimum
- **Tablet**: 1200 x 1920 pixels minimum

## Screenshot Content Suggestions

### Screenshot 1: Home Screen
- Show the main dashboard with prayer times
- Highlight the clean, Islamic design
- Display current location (Makkah/Madinah)

### Screenshot 2: Hajj/Umrah Guide
- Show the step-by-step guide interface
- Display Arabic text with translations
- Highlight the progress tracking

### Screenshot 3: Qibla Compass
- Show the compass in action
- Display accurate direction to Kaaba
- Show location-based accuracy

### Screenshot 4: Dua Collection
- Display the categorized duas
- Show Arabic, transliteration, and translation
- Highlight the comprehensive collection

### Screenshot 5: Holy Places
- Show the interactive map or place details
- Display historical information
- Show high-quality images

## Design Guidelines

### Text Overlays
- Use readable fonts (minimum 16pt)
- High contrast colors
- Avoid covering important UI elements
- Keep text concise and impactful

### Branding
- Consistent color scheme (#1B5E20 green)
- Islamic design elements
- Professional appearance
- Clear app name visibility

### Localization
- Create screenshots in primary languages
- Arabic text should be properly displayed
- RTL layout for Arabic screenshots

## Technical Requirements

### File Formats
- PNG or JPEG
- RGB color space
- No transparency for backgrounds
- High resolution (2x or 3x)

### Content Guidelines
- No misleading information
- Actual app interface only
- No competitor references
- Family-friendly content
